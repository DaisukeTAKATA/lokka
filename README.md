Pyha
====

Ruby Blog/CMS for Cloud.

INSTALLATION
------------

    $ gem install bundler --pre
    $ git clone git://github.com/komagata/pyha.git
    $ cd pyha
    $ rake install
    $ rackup

Access to http://localhost:9292/
