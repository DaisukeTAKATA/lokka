はじめよう

Windowsの場合

  1. RubyInstaller for Windowsをインストールします。
  2. Pyhaをダウンロードして解凍します。
  3. setup.exeを実行します。
  4. pyha.exeを実行するとローカル環境にWebサーバーが立ち上がるのでhttp://localhost:9393/にアクセスしてください。
  5. 投稿はhttp://localhost:9393/admin/から行ないます。初期ユーザー / パスワードはtest / testです。（実際に公開する場合は必ずユーザー / パスワードを変更してください）

Mac OS X, UNIXの場合

  1. Pyhaをダウンロードして解凍します。
  2. install.rbを実行します。
  3. pyha.rbを実行するとローカル環境にWebサーバーが立ち上がるのでhttp://localhost:9292/にアクセスしてください。
  4. 投稿はhttp://localhost:9292/admin/から行ないます。初期ユーザー／パスワードはtest / testです。（実際に公開する場合は必ずユーザー／パスワードを変更してください）
